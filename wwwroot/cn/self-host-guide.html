独立部署(私有化部署) 

子集化服务支持独立部署(私有化部署)————将子集化服务的核心库部署在客户自己的服务器集群上。
  • 如果对稳定性和安全性有较高的要求，则需要将字体服务部署到自己的服务器上；
  • 独立部署服务可实现高可用性，响应速度达到毫秒级，可支持高并发高负载；
  • 独立部署服务可支持负载均衡、支持K8S等；
  • 部署完成后，字体服务就会完全独立运行；
  • 部署完成后，可以确保不会在任何时候被外网访问（用户可以将安全策略设置为拒绝外网连接）。

部署文档- 我们支持以下方式实现快速部署

  • docker-composer(推荐)
  • docker
  • ubuntu
  • Mac OS
  • Windows
  • CentOS

请根据情况选择适合的方式进行部署。

配置文件

linux下 配置文件路径为：/etc/webfont/conf/config.yml
mac下 配置文件路径为：/usr/local/etc/webfont/conf/config.yml
Docker下请用docker run -it 命令进入容器后在/etc/webfont/conf/查看

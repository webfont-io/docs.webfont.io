# WebFont PHP SDK Quick Start Example


## Prerequisites

This requires `php` >= 5.5, `pecl`, `composer`, `grpc.so`, `crul.so`

## Install extension

 - Install the `grpc` extension

   ```sh
   $ [sudo] pecl install grpc
   ```
- Install the `crul` extension

  ```sh
  $ [sudo] pecl install crul
  ```

## Try

 - Run the demo

   ```
   $ cd grpc-client/
   $ php WebFontDemo.php
   ```

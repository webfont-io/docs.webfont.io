## ubuntu下webfont独立部署安装

## 下载二进制
```sh
ubuntu版请联系技术人员，获取安装配置文件。技术人员联系方式：+86 13774423414
```



## 授予运行权限
```sh
sudo chmod u+x ./webfont-ubuntu-amd64
```

## 复制
```sh
sudo mv ./webfont-ubuntu-amd64 /usr/local/bin/webfont-api
```

## 运行
```sh
webfont-api
```


## 配置
    配置文件在 /etc/webfont/conf/config.yml
